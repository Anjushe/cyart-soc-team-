# Week 3 - SOC Operations and Incident Handling

This folder contains documentation, evidence, and screenshots related to
Week 3 Security Operations Center (SOC) practical activities.

## Activities Covered
- Advanced log analysis using Elastic Security
- Threat intelligence integration with AlienVault OTX and Wazuh
- Alert triage and IOC validation
- Incident escalation using TheHive
- Evidence collection and preservation using Velociraptor
- Full SOC workflow simulation including attack detection and response

## Folder Overview
- Documentation: Contains the detailed SOC report and notes
- Evidence: Contains collected forensic artifacts and validation outputs
- Screenshots: Contains screenshots supporting analysis and investigation

All tasks were performed in a controlled lab environment for learning
and training purposes.
